# Ecommerce-Customer-Retention-Analysis
End-to-end data analytics project using SQL, Power BI, Excel to analyze customer retention, sales performance, and business KPIs.

## Project Overview

This project analyzes an e-commerce marketplace dataset to understand customer behavior, sales performance, product trends, and operational efficiency.

Using SQL, Excel, and Power BI, the project transforms raw transactional data into actionable business insights through data cleaning, KPI analysis, interactive dashboards, and business recommendations.

## Business Problem

E-commerce companies generate large amounts of transactional data, but without proper analysis it is difficult to understand customer purchasing behavior, identify revenue opportunities, and improve operational performance.

This project aims to answer key business questions related to customer retention, sales trends, product performance, and customer satisfaction to support data-driven decision making.
